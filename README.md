# 全国疫情数据可视化项目

## 项目简介

这是一个基于ECharts的全国疫情数据可视化项目，通过交互式地图展示各省市疫情确诊数据分布情况。项目采用响应式设计，支持多种设备访问。

## 功能特性

### 🗺️ 核心功能
- **全国疫情地图**：使用颜色深浅直观展示各省份确诊人数
- **交互式操作**：支持地图缩放、拖拽浏览
- **省份跳转**：点击地图省份区域可跳转到对应省份详情页面
- **数据提示**：鼠标悬停显示详细确诊数据

### 📊 数据展示
- 实时数据统计面板
- 分级色彩可视化
- 包含港澳台地区完整数据
- 响应式设计适配各种屏幕

### 🎨 界面特色
- 现代化深色主题设计
- 平滑动画过渡效果
- 直观的数据可视化
- 清晰的视觉层次

## 技术栈

- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **可视化库**：ECharts 5.4.3
- **地图数据**：中国地图JSON
- **样式特性**：
  - CSS Grid & Flexbox 布局
  - 渐变背景和毛玻璃效果
  - 响应式媒体查询
  - 平滑过渡动画

## 使用说明

### 基本操作
1. **查看全国数据**：打开主页面查看全国疫情分布
2. **交互探索**：
   - 鼠标悬停：查看省份确诊人数
   - 点击省份：跳转到该省份详情页面
   - 滚轮缩放：放大/缩小地图
   - 拖拽移动：调整地图显示区域

3. **数据解读**：
   - 红色越深表示确诊人数越多
   - 统计面板显示关键疫情指标
   - 图例说明不同颜色对应的数据范围

### 省份页面跳转
项目预设了完整的省份页面跳转功能：
- 点击地图任意省份
- 系统自动匹配对应的HTML文件
- 跳转到该省份的疫情详情页面

## 数据说明

### 数据来源
- 模拟数据，基于历史疫情数据模式生成
- 数据截止时间：2021-08-18 10:53:30
- 包含34个省级行政区数据

### 数据分类
确诊人数分为6个等级：
- 1-99人
- 100-999人  
- 1000-9999人
- 10000-49999人
- 50000-100000人
- 100000+人

## 自定义配置

### 修改数据
在 `index.html` 中找到 `provinceData` 数组，按以下格式更新数据：
```javascript
{
    name: '省份名称',
    value: 确诊人数
}
```

### 调整可视化样式
修改 `visualMap` 配置来调整颜色方案：
```javascript
visualMap: {
    pieces: [
        {min: 1, max: 99, label: '1-99人', color: '#颜色代码'},
        // ... 其他区间
    ]
}
```

### 添加新功能
在ECharts配置的 `series` 中添加新系列，或在事件监听中添加新交互。

## 常见问题

### Q: 地图显示不完整或空白
A: 检查网络连接，确保ECharts库正确加载。如问题持续，尝试刷新页面。

### Q: 点击省份没有跳转
A: 
1. 确保对应省份的HTML文件存在
2. 检查浏览器控制台是否有错误信息
3. 确认文件名与省份名称匹配规则正确

### Q: 移动端显示异常
A: 项目采用响应式设计，如遇显示问题请检查：
- 浏览器是否支持CSS Grid和Flexbox
- 视口设置是否正确
- 屏幕尺寸是否过小

### Q: 数据更新
A: 手动更新 `provinceData` 数组中的数据，或对接后端API实现动态数据加载。
